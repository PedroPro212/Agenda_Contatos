# Agenda_Contatos
Uma aplicação Web para gerenciar seus contatos de forma dinâmica.

## Propósito do projeto
Aprender e desenvolver a minha versão do Google Contatos

## Tecnologias utilizadas
- HTML5
- CSS3
- Bootstrap
- JavaScript
- PHP
- MySql
